Welcome To Yeogurt v1.0.0

An Automated Front-End Build Tool

Features

  • Compile SASS files
  • Optimise images using OptiPNG and JPEGTran
  • Built-in Preview Server
  • Automatically lints JavaScript and CSS
  • JavaScript Unit Testing with Mocha
Go to dashboard

Want to skip this page and go right to the dashboard from now on?

Just open up Gruntfile.js in the yeogurt/sites/example-site folder and change line 124 from this: 
path: 'http://localhost:<%= connect.options.port %>/'

To this: 
path: 'http://localhost:<%= connect.options.port %>/dashboard/index.html'
